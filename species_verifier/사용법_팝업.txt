💡 국립수산과학원 학명 검증기 사용 안내 💡

■ 기본 사용법:
   ○ 탭 선택: 검증할 생물 종류(해양생물, 미생물, 담수 등 전체생물)에 맞는 탭을 선택하세요.
   
   ○ 직접 입력 (실시간 처리):
     - 텍스트 상자에 학명을 입력합니다 (쉼표로 학명 구분)
     - 최대 10개까지 입력 가능 (실시간 빠른 처리)
     - 입력 후 '검증' 버튼 클릭
     - 특징: 10개 이하 → 즉시 실시간 처리로 빠르게 결과 확인
   
   ○ 파일 입력 (배치 처리):
     - CSV, XLSX, TXT 파일 사용 (첫 열에 이름)
     - '찾기' 버튼으로 파일 선택 (최대 3000개)
     - '검증' 버튼 클릭
     - 특징: 대용량 파일을 100개씩 배치로 나누어 안전하게 처리하며, 
       결과는 실시간으로 표시됩니다 (배치간 3초 지연)

■ 결과 확인 및 활용:
   ○ 결과는 하단 목록에 실시간으로 표시됩니다
   
   ○ 결과 복사 (🖱️ 오른쪽 클릭):
     - 특정 셀: '{컬럼명}' 내용 복사 선택
     - 행 전체: 선택 행 전체 정보 복사 선택
   
   ○ 추가 기능:
     - URL 열기: URL 셀 더블 클릭
     - 위키 요약 보기: 위키백과 요약 셀 더블 클릭
     - 결과 내보내기: 결과 목록 오른쪽 클릭 > '전체 결과 Excel로 저장' 
       또는 하단 '저장' 버튼 클릭

■ 파일 캐시 관리 🔄:
   ○ 파일 로드 후 텍스트 입력: 파일을 로드한 후 텍스트 입력란에 새로운 학명을 
     입력해도 여전히 파일 데이터로 검증됩니다
   
   ○ 캐시 삭제 시점:
     - 검증 완료 후: 파일 캐시가 자동으로 삭제되어 이후 텍스트 입력만 사용됩니다
     - 검증 취소 후: 모든 탭의 파일 캐시가 삭제됩니다
   
   ○ 재사용: 파일로 다시 검증하려면 "파일 찾기" 버튼을 다시 눌러 파일을 로드하세요

■ 처리 시간 안내 ⏱️:
   ○ 실시간 처리 (직접 입력):
     - 10개 이하: 약 10-30초 (즉시 처리, 지연 시간 최소화)
     - 개별 항목 당 약 0.5-1초로 빠른 처리
   
   ○ 배치 처리 (파일 입력):
     - 안정성을 위해 100개씩 배치로 나누어 처리
     - 100개: 약 2-3분
     - 1000개: 약 15-35분  
     - 3000개: 약 45분-1시간 45분
   
   ○ 추가 정보:
     - 실시간 지연: 0.5초 (빠른 처리용)
     - 배치 지연: 배치 간 3초 대기 시간 적용 (API 서버 보호)
     - 취소 기능: 처리 중 언제든지 '취소' 버튼으로 중단 가능
     - 참고: 처리 시간은 네트워크 상태와 서버 응답 속도에 따라 달라질 수 있습니다

■ 오류 및 팁:
   ○ 네트워크 오류: 특정 항목 검증 실패 시, 해당 셀을 오른쪽 클릭하여 복사 후 
     잠시 뒤 다시 시도해 보세요
   ○ 입력 제한: 직접 입력 10개, 파일 입력 3000개 초과 시 나누어 입력하세요
   ○ 처리 방식 선택:
     - 빠른 확인: 10개 이하 → 직접 입력 (실시간 처리)
     - 대량 처리: 11개 이상 → 파일 입력 (배치 처리)
